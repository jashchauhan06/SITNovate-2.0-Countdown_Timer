# 24 Hour Hackathon Countdown Timer

A React-based countdown timer with flip card animations and admin controls.

## Installation

```bash
npm install
```

## Running the App

```bash
npm start
```

The app will open at `http://localhost:3000`

## Routes

- `/` - Public countdown display
- `/admin` - Admin control panel

## Features

- Flip card animations for each digit
- Start/Stop/Reset controls in admin panel
- Real-time synchronization using localStorage
- Responsive design
